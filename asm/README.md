#  ASM

ASM是业界流行的java字节码操作和解析框架.它可以用于修改现有类或直接以二进制形式动态生成类.



学ASM的目的：重写`HashMap`中的`hash`方法,增加hash冲突的次数,方便调试程序


`hash`已经重写了,但是使用类加载器进行分加载的时候,被拒绝了,不允许加载以`java.`开头的类
